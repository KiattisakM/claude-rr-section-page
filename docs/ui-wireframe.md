## Responsive Breakpoints

- **Desktop**: 1200px and above - 4 cards in a row
- **Tablet**: 768px - 992px - 2 cards in a row
- **Mobile**: Below 768px - 1 card in a row, stacked vertically

## Interactive States

### Buttons
- **Default**: Outlined or filled background
- **Hover**: Subtle shadow or color change
- **Active**: Pressed state with different background

### Cards
- **Default**: Clean border with shadow
- **Hover**: Elevated shadow, slight scale
- **Active**: Pressed state for mobile

### Input Fields
- **Default**: Border with placeholder
- **Focus**: Highlighted border with icon
- **Filled**: Validated state with content
